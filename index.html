YouCompleteMe Installation

Posted on 2018-09-03 | In Configuration

Installation

Install development tools and CMake:

yum install build-essential cmake

Make sure you have Python headers installed:

yum install python-devel rh-python36-python-devel.x86_64

If python3 has not been installed on your CentOS:
    Enable Software Collections
        sudo yum install centos-release-scl
    Install Python3
        sudo yum install rh-python36

If you want to use python 3.6 as default
    scl enable rh-python36 bash


Compiling YCM with semantic support for C-family languages:

cd ~/.vim/bundle/YouCompleteMe
./install.py --clang-completer

Compiling YCM without semantic support for C-family languages:

cd ~/.vim/bundle/YouCompleteMe
./install.py

cd ~/.vim/bundle/YouCompleteMe
./install.py --all
